# OpenHotel-OS

**OpenHotel OS** is an open-source, multi-platform hospitality TV platform designed to deliver a fast, modern, and fully customizable guest experience.

Built for hotels, apartments, and smart rooms, it transforms any TV or set-top box into a smart guest hub — combining a high-performance native interface, a flexible web-based system, and a powerful remote management dashboard.

---

## **Overview**

OpenHotel OS focuses on **flexibility, performance, and real-world usability**.

It introduces a hybrid architecture where a native TV app works alongside a WebView-based UI, allowing both speed and dynamic customization without constant updates.

From the first “Hello” screen to daily guest interactions, the system is designed to feel smooth, simple, and premium.

---

## **Key Features**

### **Client (TV / STB / Android)**

####  **User Experience**
- Simple and customizable UI
- Smooth and easy navigation
- Hotel facilities, rules, and information
- Greetings from hotel staff

#### **Multilingual System**
- System-wide language support
- Apple-style **Hello Experience** for language selection
- Dynamic language switching across UI and content

#### **Media & Entertainment**
- Built-in IPTV player / TV tuner API
- Streaming and casting support (planned)

#### **Hybrid UI System**
- **App Mode (Native UI)**
- **WebView Mode (HTML-based UI)**

#### **Guest Services**
- Food and service ordering
- Guest inbox (announcements, updates)
- Explore city & staycation ideas

#### **External Device Support**
- Optional **HDMI / Input access**
- Device-aware (Google TV / Android TV supported)
- Smart detection and switching (when available)

#### **System Features**
- Sleep mode (prevents screen burn-in)
- Offline-capable operation
- UI templates for easy setup
- Drag & drop UI design (planned)
- Multi-OS support (upcoming)
